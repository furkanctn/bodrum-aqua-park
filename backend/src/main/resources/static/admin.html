Yeni kart tanımla

Kart okuyucunun verdiği UID ile aynı biçimde girin (veya manuel). Aynı UID iki kez eklenemez. Okuyucu bağlantı testi (HID / seri ayırımı için).

Fiş yazıcı testi (Sewoo / ESC-POS)

Windows: Aygıt Yöneticisi → Bağlantı noktaları (COM ve LPT) → COM numarası. Mac: Terminal’de ls /dev/cu.* — yazıcı takılıyken çıkan ad (örn. /dev/cu.usbserial-…) genelde doğru porttur; listede yoksa aşağıya elle yazın. Yazı çıkmazsa baud’u 115200 veya 9600 deneyin. Hâlâ boşsa: yazıcı USB’sini çıkarıp ls /dev/cu.* tekrar — kaybolan satır gerçekten bu yazıcıdır; başka bir USB çubuğu (FTDI) yanlışlıkla seçilmiş olabilir. Test modunda önce minimal veya kesim yok deneyin.
